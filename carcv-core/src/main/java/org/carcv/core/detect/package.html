Provides an abstraction of basic detection objects. All Detectors implement the {@link Detector} interface.

These are three types of objects used for the three stages of detection:

  1. Sorting of video frames into real cars -- {@link CarSorter}
  2. Detection of {@link NumberPlate}s -- {@link NumberPlateDetector}
  3. Detection of car {@link Speed} -- {@link SpeedDetector}
