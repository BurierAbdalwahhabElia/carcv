Provides an abstraction of basic detection objects. All Detectors implement the {@link org.carcv.core.detect.Detector} interface.

These are three types of objects used for the three stages of detection:

  1. Sorting of video frames into real cars -- {@link org.carcv.core.detect.CarSorter}
  2. Detection of {@link org.carcv.core.model.NumberPlate}s -- {@link org.carcv.core.detect.NumberPlateDetector}
  3. Detection of car {@link org.carcv.core.model.Speed} -- {@link org.carcv.core.detect.SpeedDetector}
